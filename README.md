# AI Sports Talent Scout

An end-to-end Machine Learning project that identifies promising cricket players and predicts their future potential and estimated auction value using advanced analytics and predictive modeling.

---

## Project Overview

AI Sports Talent Scout is designed to assist cricket franchises, scouts, and analysts in identifying future stars based on player performance metrics.

The system analyzes batting, bowling, fielding, fitness, and experience-related features to generate:

* Future Potential Classification
* Auction Value Prediction
* Scout Rating
* Player Skill Analysis Dashboard

---

## Problem Statement

Finding future cricket stars requires extensive scouting and performance analysis.

This project uses Machine Learning to:

* Evaluate player performance
* Estimate future potential
* Predict market value
* Assist data-driven talent scouting
